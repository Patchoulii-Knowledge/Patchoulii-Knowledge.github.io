欧拉第一积分

欧拉第一积分就是 B\text{B}B 函数，读作Beta函数（贝塔函数）

定义

B(a,b)=∫01xa−1(1−x)b−1dx\text{B}\left( a,b \right) =\int_0^1{x^{a-1}\left( 1-x \right) ^{b-1}\text{d}x} B(a,b)=∫01​xa−1(1−x)b−1dx

性质及其证明

1. B(a,b)=B(b,a)\text{B}\left( a,b \right) =\text{B}\left( b,a \right)B(a,b)=B(b,a)

B(a,b)=B(b,a)\text{B}\left( a,b \right) =\text{B}\left( b,a \right) B(a,b)=B(b,a)

证 直接作代换 x→1−xx\rightarrow1-xx→1−x 即可

B(a,b)=∫01xa−1(1−x)b−1dx=x→1−x∫10(1−x)a−1xb−1d(1−x)\text{B}\left( a,b \right) =\int_0^1{x^{a-1}\left( 1-x \right) ^{b-1}\text{d}x}\xlongequal{x\rightarrow 1-x}\int_1^0{\left( 1-x \right) ^{a-1}x^{b-1}\text{d}\left( 1-x \right)} B(a,b)=∫01​xa−1(1−x)b−1dxx→1−x∫10​(1−x)a−1xb−1d(1−x)

=∫10(1−x)a−1xb−1(−dx)=∫01(1−x)a−1xb−1dx=B(b,a)=\int_1^0{\left( 1-x \right) ^{a-1}x^{b-1}\left( -\text{d}x \right)}=\int_0^1{\left( 1-x \right) ^{a-1}x^{b-1}\text{d}x}=\text{B}\left( b,a \right) =∫10​(1−x)a−1xb−1(−dx)=∫01​(1−x)a−1xb−1dx=B(b,a)

2. B(a,b)=a−1a+b−1B(1−a,b)\text{B}\left( a,b \right) =\frac{a-1}{a+b-1}\text{B}\left( 1-a,b \right)B(a,b)=a+b−1a−1​B(1−a,b)

B(a,b)=a−1a+b−1B(1−a,b)   (a>1)\text{B}\left( a,b \right) =\frac{a-1}{a+b-1}\text{B}\left( 1-a,b \right) \ \ \ \left( a>1 \right) B(a,b)=a+b−1a−1​B(1−a,b)   (a>1)

证 分部积分+裂项即可

B(a,b)=∫01xa−1(1−x)b−1dx=∫01(1−x)b−1d(xaa)=xa(1−x)b−1a∣01+b−1a∫01xa(1−x)b−2dx=b−1a∫01(1−(1−x))⋅xa−1(1−x)b−2dx=b−1a[∫01xa−1(1−x)b−2dx−∫01xa−1(1−x)b−1dx]=b−1a[B(a,b−1)−B(a,b)]\begin{aligned} \text{B}\left( a,b \right) &=\int_0^1{x^{a-1}\left( 1-x \right) ^{b-1}\text{d}x}=\int_0^1{\left( 1-x \right) ^{b-1}\text{d}\left( \frac{x^a}{a} \right)}\\ &=\left. \frac{x^a\left( 1-x \right) ^{b-1}}{a} \right|_{0}^{1}+\frac{b-1}{a}\int_0^1{x^a\left( 1-x \right) ^{b-2}\text{d}x}\\ &=\frac{b-1}{a}\int_0^1{\left( 1-\left( 1-x \right) \right) \cdot x^{a-1}\left( 1-x \right) ^{b-2}\text{d}x}\\ &=\frac{b-1}{a}\left[ \int_0^1{x^{a-1}\left( 1-x \right) ^{b-2}\text{d}x}-\int_0^1{x^{a-1}\left( 1-x \right) ^{b-1}\text{d}x} \right]\\ &=\frac{b-1}{a}\left[ \text{B}\left( a,b-1 \right) -\text{B}\left( a,b \right) \right]\\ \end{aligned} B(a,b)​=∫01​xa−1(1−x)b−1dx=∫01​(1−x)b−1d(axa​)=axa(1−x)b−1​∣∣∣∣∣​01​+ab−1​∫01​xa(1−x)b−2dx=ab−1​∫01​(1−(1−x))⋅xa−1(1−x)b−2dx=ab−1​[∫01​xa−1(1−x)b−2dx−∫01​xa−1(1−x)b−1dx]=ab−1​[B(a,b−1)−B(a,b)]​

∴B(a,b)=b−1a+b−1B(a,b−1)\therefore \text{B}\left( a,b \right) =\frac{b-1}{a+b-1}\text{B}\left( a,b-1 \right) ∴B(a,b)=a+b−1b−1​B(a,b−1)

3. B(m,n)=(n−1)!(m−1)!(m+n−1)!\text{B}\left( m,n \right) =\frac{\left( n-1 \right) !\left( m-1 \right) !}{\left( m+n-1 \right) !}B(m,n)=(m+n−1)!(n−1)!(m−1)!​

B(m,n)=(n−1)!(m−1)!(m+n−1)! (m,n∈N∗)\text{B}\left( m,n \right) =\frac{\left( n-1 \right) !\left( m-1 \right) !}{\left( m+n-1 \right) !}\ \left( m,n\in \mathbb{N}^* \right) B(m,n)=(m+n−1)!(n−1)!(m−1)!​ (m,n∈N∗)

证 一直用上面的性质

B(m,n)=n−1m+n−1⋅n−2m+n−2⋯1m+1⋅B(m,1)\text{B}\left( m,n \right) =\frac{n-1}{m+n-1}\cdot \frac{n-2}{m+n-2}\cdots \frac{1}{m+1}\cdot \text{B}\left( m,1 \right) B(m,n)=m+n−1n−1​⋅m+n−2n−2​⋯m+11​⋅B(m,1)

我们知道

B(m,1)=∫01xm−1dx=1m\text{B}\left( m,1 \right) =\int_0^1{x^{m-1}\text{d}x}=\frac{1}{m} B(m,1)=∫01​xm−1dx=m1​

所以

B(m,n)=n−1m+n−1⋅n−2m+n−2⋯1m+1⋅1m=(n−1)!(m−1)!(m+n−1)!\text{B}\left( m,n \right) =\frac{n-1}{m+n-1}\cdot \frac{n-2}{m+n-2}\cdots \frac{1}{m+1}\cdot \frac{1}{m}=\frac{\left( n-1 \right) !\left( m-1 \right) !}{\left( m+n-1 \right) !} B(m,n)=m+n−1n−1​⋅m+n−2n−2​⋯m+11​⋅m1​=(m+n−1)!(n−1)!(m−1)!​

4. B(a,b)=∫0∞ya−1(1+y)a+b\text{B}\left( a,b \right) =\int_0^{\infty}{\frac{y^{a-1}}{\left( 1+y \right) ^{a+b}}}B(a,b)=∫0∞​(1+y)a+bya−1​

B(a,b)=∫0∞ya−1(1+y)a+b\text{B}\left( a,b \right) =\int_0^{\infty}{\frac{y^{a-1}}{\left( 1+y \right) ^{a+b}}} B(a,b)=∫0∞​(1+y)a+bya−1​

证 作代换代换 x=y1+yx=\frac{y}{1+y}x=1+yy​ 即可：

x=y1+y⇒y=x1−xx=\frac{y}{1+y}\Rightarrow y=\frac{x}{1-x} x=1+yy​⇒y=1−xx​

B(a,b)=∫01xa−1(1−x)b−1dx=x→y1+y∫0∞(y1+y)a−1(11+y)b−1d(y1+y)\text{B}\left( a,b \right) =\int_0^1{x^{a-1}\left( 1-x \right) ^{b-1}\text{d}x}\xlongequal{x\rightarrow \frac{y}{1+y}}\int_0^{\infty}{\left( \frac{y}{1+y} \right) ^{a-1}\left( \frac{1}{1+y} \right) ^{b-1}\text{d}\left( \frac{y}{1+y} \right)} B(a,b)=∫01​xa−1(1−x)b−1dxx→1+yy​∫0∞​(1+yy​)a−1(1+y1​)b−1d(1+yy​)

=∫0∞(y1+y)a−1(11+y)b−1(1(1+y)2dy)=∫0∞ya−1(1+y)a+bdy=\int_0^{\infty}{\left( \frac{y}{1+y} \right) ^{a-1}\left( \frac{1}{1+y} \right) ^{b-1}\left( \frac{1}{\left( 1+y \right) ^2}\text{d}y \right)}=\int_0^{\infty}{\frac{y^{a-1}}{\left( 1+y \right) ^{a+b}}\text{d}y} =∫0∞​(1+yy​)a−1(1+y1​)b−1((1+y)21​dy)=∫0∞​(1+y)a+bya−1​dy

5. B(a,1−a)=πsin⁡(aπ)(0<a<1)\text{B}\left( a,1-a \right) =\frac{\pi}{\sin \left( a\pi \right)}\left( 0<a<1 \right)B(a,1−a)=sin(aπ)π​(0<a<1)

B(a,1−a)=πsin⁡(aπ)(0<a<1) ⇒B(12,12)=π\text{B}\left( a,1-a \right) =\frac{\pi}{\sin \left( a\pi \right)}\left( 0<a<1 \right) \ \Rightarrow \text{B}\left( \frac{1}{2},\frac{1}{2} \right) =\pi B(a,1−a)=sin(aπ)π​(0<a<1) ⇒B(21​,21​)=π

证 根据第 444 个性质，可以化成如下形式：

B(a,1−a)=∫0∞ya−11+ydy\text{B}\left( a,1-a \right) =\int_0^{\infty}{\frac{y^{a-1}}{1+y}\text{d}y} B(a,1−a)=∫0∞​1+yya−1​dy

欧拉第二积分

欧拉第二积分就是著名的 Γ\GammaΓ 函数，读作Gamma函数（伽玛函数）

定义

Γ(z)=∫0∞xz−1e−xdx\Gamma \left( z \right) =\int_0^{\infty}{x^{z-1}e^{-x}\text{d}x} Γ(z)=∫0∞​xz−1e−xdx

性质及其证明

1.递推公式

Γ(n+1)=n!(n∈N)\Gamma \left( n+1 \right) =n!\left( n\in \mathbb{N} \right) Γ(n+1)=n!(n∈N)

Γ(z+1)=zΓ(z)  (Rez>0)\Gamma \left( z+1 \right) =z\Gamma \left( z \right) \,\,\left( \text{Re}z>0 \right) Γ(z+1)=zΓ(z)(Rez>0)

证 应用分部积分法

Γ(z+1)=∫0∞xze−xdx=−∫0∞xzd(e−x)=[−xze−x]∣0+∞+z∫0∞e−xd(xz)=0+z∫0∞e−xxz−1dx=zΓ(z)\begin{aligned} \Gamma \left( z+1 \right) &=\int_0^{\infty}{x^ze^{-x}\text{d}x}=-\int_0^{\infty}{x^z\text{d}\left( e^{-x} \right)}\\ &=\left[ -x^ze^{-x} \right] \mid _{0}^{+\infty}+z\int_0^{\infty}{e^{-x}\text{d}\left( x^z \right)}\\ &=0+z\int_0^{\infty}{e^{-x}x^{z-1}\text{d}x}\\ &=z\Gamma \left( z \right)\\ \end{aligned} Γ(z+1)​=∫0∞​xze−xdx=−∫0∞​xzd(e−x)=[−xze−x]∣0+∞​+z∫0∞​e−xd(xz)=0+z∫0∞​e−xxz−1dx=zΓ(z)​

∵Γ(1)=∫0∞e−xdx=1∴Γ(n+1)=nΓ(n)=n⋅(n−1)⋅Γ(n−1)=⋯=n!\begin{aligned} \because \Gamma \left( 1 \right) &=\int_0^{\infty}{e^{-x}\text{d}x}=1\\ \therefore \Gamma \left( n+1 \right) &=n\Gamma \left( n \right)\\ &=n\cdot \left( n-1 \right) \cdot \Gamma \left( n-1 \right)\\ &=\cdots\\ &=n!\\ \end{aligned} ∵Γ(1)∴Γ(n+1)​=∫0∞​e−xdx=1=nΓ(n)=n⋅(n−1)⋅Γ(n−1)=⋯=n!​

所以，我们可以把Γ\GammaΓ函数看成阶乘的推广

2.常用性质

Γ(z)=∫0∞xz−1e−xdx=kz∫0∞xz−1e−kxdx  (Rez>0)\Gamma \left( z \right) =\int_0^{\infty}{x^{z-1}e^{-x}\text{d}x}=k^z\int_0^{\infty}{x^{z-1}e^{-kx}\text{d}x}\,\,\left( \text{Re}z>0 \right) Γ(z)=∫0∞​xz−1e−xdx=kz∫0∞​xz−1e−kxdx(Rez>0)

证 这个就很好证了，直接作代换 x→k⋅xx\rightarrow k\cdot xx→k⋅x

Γ(z)=∫0∞xz−1e−xdx=x→k⋅x∫0∞(k⋅x)z−1e−k⋅xd(k⋅x)\Gamma \left( z \right) =\int_0^{\infty}{x^{z-1}e^{-x}\text{d}x}\xlongequal{x\rightarrow k\cdot x}\int_0^{\infty}{\left( k\cdot x \right) ^{z-1}e^{-k\cdot x}\text{d}\left( k\cdot x \right)} Γ(z)=∫0∞​xz−1e−xdxx→k⋅x∫0∞​(k⋅x)z−1e−k⋅xd(k⋅x)

=∫0∞(xz−1kz−1)e−kx(kdx)=kz∫0∞xz−1e−kxdx=\int_0^{\infty}{\left( x^{z-1}k^{z-1} \right) e^{-kx}\left( k\text{d}x \right)}=k^z\int_0^{\infty}{x^{z-1}e^{-kx}\text{d}x} =∫0∞​(xz−1kz−1)e−kx(kdx)=kz∫0∞​xz−1e−kxdx

3. Γ\GammaΓ函数和B\text{B}B函数的联系

这个非常重要！！一定要记住！！（敲黑板，敲黑板）

B(a,b)=Γ(a)⋅Γ(b)Γ(a+b)\text{B}\left( a,b \right) =\frac{\Gamma \left( a \right) \cdot \Gamma \left( b \right)}{\Gamma \left( a+b \right)} B(a,b)=Γ(a+b)Γ(a)⋅Γ(b)​

证 利用性质2，把xxx换成t⋅xt\cdot xt⋅x：

Γ(a)ta=∫0∞xa−1e−txdx\frac{\Gamma \left( a \right)}{t^a}=\int_0^{\infty}{x^{a-1}e^{-tx}\text{d}x} taΓ(a)​=∫0∞​xa−1e−txdx

然后呢，继续做代换，代换a→a+b(b>0),t→t+1a \rightarrow a+b (b>0),t \rightarrow t+1a→a+b(b>0),t→t+1，我们可以得出：

Γ(a+b)(t+1)a+b=∫0∞xa+b−1e−(t+1)xdx\frac{\Gamma \left( a+b \right)}{\left( t+1 \right) ^{a+b}}=\int_0^{\infty}{x^{a+b-1}e^{-\left( t+1 \right) x}\text{d}x} (t+1)a+bΓ(a+b)​=∫0∞​xa+b−1e−(t+1)xdx

接下来呢，两边乘上ta−1t^{a-1}ta−1，并同时从 000 到 ∞\infty∞ 对ttt积分，（注意，是对 ttt 积分！）：

Γ(a+b)∫0∞ta−1(t+1)a+b=∫0∞ta−1dt∫0∞xa+b−1e−(t+1)xdx\Gamma \left( a+b \right) \int_0^{\infty}{\frac{t^{a-1}}{\left( t+1 \right) ^{a+b}}}=\int_0^{\infty}{t^{a-1}\text{d}t}\int_0^{\infty}{x^{a+b-1}e^{-\left( t+1 \right) x}\text{d}x} Γ(a+b)∫0∞​(t+1)a+bta−1​=∫0∞​ta−1dt∫0∞​xa+b−1e−(t+1)xdx

然后观察这个式子：

左边：我擦，这不就是贝塔函数吗？（见β\betaβ函数的性质4）

右边：我擦，我似乎可以重新配啊。

于是乎：

Γ(a+b)⋅B(a,b)=∫0∞xa+b−1e−ydx∫0∞ta−1e−tydt=∫0∞xa+b−1e−y⋅Γ(a)xadx=Γ(a)∫0∞xb−1e−ydx=Γ(a)⋅Γ(b)\begin{aligned} \Gamma \left( a+b \right) \cdot \text{B}\left( a,b \right) &=\int_0^{\infty}{x^{a+b-1}e^{-y}\text{d}x\int_0^{\infty}{t^{a-1}e^{-ty}\text{d}t}}\\ &=\int_0^{\infty}{x^{a+b-1}e^{-y}\cdot \frac{\Gamma \left( a \right)}{x^a}\text{d}x}=\Gamma \left( a \right) \int_0^{\infty}{x^{b-1}e^{-y}\text{d}x}=\Gamma \left( a \right) \cdot \Gamma \left( b \right)\\ \end{aligned} Γ(a+b)⋅B(a,b)​=∫0∞​xa+b−1e−ydx∫0∞​ta−1e−tydt=∫0∞​xa+b−1e−y⋅xaΓ(a)​dx=Γ(a)∫0∞​xb−1e−ydx=Γ(a)⋅Γ(b)​

接下来我们移项就好啦！

B(a,b)=Γ(a)⋅Γ(b)Γ(a+b)\text{B}\left( a,b \right) =\frac{\Gamma \left( a \right) \cdot \Gamma \left( b \right)}{\Gamma \left( a+b \right)} B(a,b)=Γ(a+b)Γ(a)⋅Γ(b)​

如果你看懂了的话，你会发现这个方法实在是秒啊！

4.余元公式

Γ(p)Γ(1−p)=πsin⁡(pπ), p∈(0,1)⇒Γ(12)=π\Gamma \left( p \right) \Gamma \left( 1-p \right) =\frac{\pi}{\sin \left( p\pi \right)},\ p\in \left( 0,1 \right) \Rightarrow \Gamma \left( \frac{1}{2} \right) =\sqrt{\pi} Γ(p)Γ(1−p)=sin(pπ)π​, p∈(0,1)⇒Γ(21​)=π​

证 直接用上面的性质：

Γ(a)⋅Γ(1−a)=B(a,1−a)⋅Γ(1)=B(a,1−a)=πsin⁡ aπ(0<a<1)\Gamma \left( a \right) \cdot \Gamma \left( 1-a \right) =\text{B}\left( a,1-a \right) \cdot \Gamma \left( 1 \right) =\text{B}\left( a,1-a \right) =\frac{\pi}{\sin\text{\ }a\pi}\left( 0<a<1 \right) Γ(a)⋅Γ(1−a)=B(a,1−a)⋅Γ(1)=B(a,1−a)=sin aππ​(0<a<1)
